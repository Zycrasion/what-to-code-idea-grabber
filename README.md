# What To Code Idea Grabber
Grabs Every Single Idea from https://what-to-code.com
I made this when i was bored soooo.

## How to setup:

run
``npm run setup``
in the console.

## How to run:

### Development

run ``npm run test``
in the console.

### Downloading
run ``npm run start``
